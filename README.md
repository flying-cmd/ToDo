# Tech

## Frontend: React

## Backend: Express

## Database: MySQL

## Run command:
1. Intasll
```
cd ToDo
npm install
```

2. Config .env file
```
PORT=3200
MYSQL_USER=your database uername
MYSQL_PASSWORD=your database password
MYSQL_DB=your database name
```

3. Open two terminal, for the first terminal
```
cd frontend
npm run build
npm run start
```

For the second terminal
```
cd backend
npm run build
npm run start
```
